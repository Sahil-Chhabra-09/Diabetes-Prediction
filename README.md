# Diabetes-Prediction

DataSet link : https://www.dropbox.com/s/uh7o7uyeghqkhoy/diabetes.csv?dl=0

DataSet has following attributes:
  * Pregnancies
  * Glucose
  * BloodPressure
  * SkinThickness	
  * Insulin	
  * BMI	
  * Diabetes Pedigree Function
  * Age
  
based on which we predict the outcome
